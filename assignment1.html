2b. Color Justification

For the color based on GPA, I decided to the RGB function to create a spectrum of red colors ranging from fully saturation (255, 0, 0) to almost zero saturation (1, 0, 0) where a high saturation corresponds to a high GPA. This is effective for visualizing because it allows the GPA data tightly bounded by the range (1.7, 4.0] to be represented by the integer range [1, 255]. In addition, I argue that red it the most vibrant of the three RGB colors, creating a greater distinction between fully saturated red to almost zero saturation red (or black).
